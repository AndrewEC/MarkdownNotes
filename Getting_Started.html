Welcome to MarkdownNotes!!!

MarkdownNotes is a single, self-contained, HTML file that provides functionality to edit and view Markdown formatted text in a Notebook esque format.

To help get you started there a few How-To pages available that explain some of the specifics of this program.

Additionally, there is a Markdown Cheat Sheet page that provides a summary of the common markdown notations that are supported by MarkdownNotes.

You can navigate to these pages by clicking the link in the previous sentence or clicking one of the links in the left-hand nav.
